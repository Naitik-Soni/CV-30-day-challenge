# CV-30-day-challenge
Repository for storing the code and data related to Computer vision 30 days challenge 

| Day    | Topic            | Scope                                                     |
|--------|------------------|-----------------------------------------------------------|
| Day-1  | CV Basics        | load, save, display, resize, rotate, flip, HSV           |
| Day-2  | Thresholding     | simple, Otsu, adaptive                                    |
| Day-3  | Filtering        | box, gaussian, median, bilateral                          |
| Day-4  | Morphology       | erosion, dilation, opening, closing                       |
| Day-5  | Edge Detection   | Sobel, Laplacian, Canny                                   |
| Day-6  | Hough Transform  | lines, circles                                            |
| Day-7  | Histograms       | histogram, HE, CLAHE                                      |
| Day-8  | Contours              | hierarchy, approxPoly, bounding boxes       |
| Day-9  | Corner Detection      | Harris, Shi-Tomasi, FAST                    |
| Day-10 | Feature Descriptors   | BRIEF, ORB descriptor                       |
| Day-11 | Feature Matching      | BFMatcher, FLANN, Lowe ratio test           |
| Day-12 | Optical Flow          | Lucas–Kanade sparse + dense                 |
| Day-13 | Object Tracking       | KCF, CSRT, MOSSE                            |
| Day-14 | Homography            | perspective transform, RANSAC               |
| Day-15 | Image Stitching       | panorama creation                           |
| Day-16 | Background Subtraction| MOG2, KNN                                   |
| Day-17 | Blob Detection        | LoG, DoG, SimpleBlobDetector                |
| Day-18 | Template Matching     | matchTemplate, multi-scale matching         |
| Day-19 | Color Segmentation    | HSV masking, color tracking                 |
| Day-20 | Image Pyramids        | Gaussian & Laplacian pyramids               |
| Day-21 | Watershed Segmentation| markers, region labeling                    |
| Day-22 | CNN Basics            | conv, padding, stride, channels             |
| Day-23 | CNN Architectures     | LeNet, AlexNet, VGG, ResNet                 |
| Day-24 | Training a CNN        | dataset → augmentation → training → eval    |
| Day-25 | Transfer Learning     | MobileNet, ResNet finetuning                |
| Day-26 | Object Detection I    | R-CNN, Fast R-CNN                           |
| Day-27 | Object Detection II   | YOLO, SSD, RetinaNet                        |
| Day-28 | Segmentation          | U-Net, Mask R-CNN                           |
| Day-29 | Pose / Keypoints DL   | heatmaps, pose estimation                   |
| Day-30 | Final Project         | build a complete CV pipeline                |
